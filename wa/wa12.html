Cat Facts

I added a user input that allows users to select the number of cat facts they would like to receive. The user can select a number between 1 and 3 using the arrows, however if they type a number larger than 3 they will receive an error message that states the parameters. The API I'm using already alowed for this functionality simply by adding ?count= to the end of the link so all I'm doing in JavaScript is changing the endpoint accordingly.
